# ISYA301 Study Materials

Welcome to your study materials folder!

Add your PDF documents, text files, images, and other study materials to this assets folder.

## Supported File Types:
- PDF documents (.pdf)
- Text files (.txt)
- Word documents (.doc, .docx)
- Excel spreadsheets (.xls, .xlsx)
- PowerPoint presentations (.ppt, .pptx)
- Images (.jpg, .png, .gif)
- Audio files (.mp3, .wav)
- Video files (.mp4)

Place your files in the `app/src/main/assets/` folder and rebuild the app.
